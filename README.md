# IT Service Desk & Support Operations Analytics

## Overview

This project delivers an end-to-end analytics solution for IT service desk operations using real-world ITSM ticket data.  
It focuses on SLA performance, backlog health, response and resolution efficiency, and operational bottlenecks.

## Tech Stack

- Python (data cleaning, automation)
- PostgreSQL (analytics schema + SQL KPIs)
- Power BI / Excel (dashboard)
- GitHub (documentation & reproducibility)

## Business Value

- Improves visibility into SLA compliance and backlog risk
- Identifies drivers of delays and ticket reopens
- Provides decision-ready dashboards for IT Operations and Management
